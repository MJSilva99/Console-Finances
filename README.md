# Console-Finances

## Description 

This is a project where I created code using JvaScript to analyse the financial records of a company and display the total months, total, average change, greatest increase in profits/losses and greatest decrease in profits/losses of the records provided.

## Project criteria met

* Calculated the total number of months included in the dataset.
* Calculated the net total amount of Profit/Losses over the entire period.
* Calculated the average of the changes in Profit/Losses over the entire period.
* Tracked the total change in Profit/Losses from month to month and calculated the average.
* Calculated the greatest increase and in Profit/Losses (date and amount) over the entire period.
* Displayed all calculations in the console log.

## Installation
No installation is required to view the website in a web browser. The website is accessible through the deployed GitPage link below or you can download the project files and open index.html in a web browser of your choice.
